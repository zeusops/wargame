# Wargame: Red Dragon

## Setting up

Create a file `settings/login.ini` with the content:

```
login="userid"
dedicated_key="key from eugen"
```

Mail <mailto:eugsupport@eugen.fr> for the dedicated key.

## Usage

```
docker compose up
```
